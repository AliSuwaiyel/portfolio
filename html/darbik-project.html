01 Project Overview

The Problem: Many Saudis struggle to maintain consistent physical activity levels despite being aware of the health benefits. Traditional fitness apps often feel monotonous, treating exercise as a chore rather than an enjoyable activity. This lack of engagement leads to declining motivation and abandoned fitness goals.

The Solution: Darbik reimagines walking as an exciting adventure by combining AI-powered object detection with gamified daily challenges. Using Core ML and Create ML, the app challenges users to find and photograph specific objects during their walks while achieving step count targets. This dual-objective approach transforms ordinary walks into engaging scavenger hunts, making fitness feel natural and fun rather than forced.

Target Audience: Health-conscious individuals in Saudi Arabia who want to increase their daily activity levels but find traditional fitness apps uninspiring. The app particularly appeals to those who enjoy games, exploration, and technology-driven experiences.

Context: Developed as part of the Apple Developer Academy AI Foundation Program at TUWAIQ, Darbik represents a groundbreaking approach to public health in Saudi Arabia. This 5-week intensive program provided hands-on experience with cutting-edge iOS development and machine learning technologies. The project addresses a critical national health priority while showcasing the power of AI to create meaningful behavioral change.

Architecture Overview

The app follows a modular MVVM (Model-View-ViewModel) architecture built with SwiftUI. The machine learning pipeline consists of custom-trained Create ML models converted to Core ML format for optimal on-device performance. The Vision framework processes camera frames in real-time, feeding them to the Core ML model for object classification. HealthKit integration runs in the background to track steps continuously, while the challenge system coordinates between detected objects and step goals to determine completion status.

04 Development Journey

Challenges Faced

Real-Time Object Detection Accuracy

Achieving reliable object recognition in varying lighting conditions and angles proved challenging. Solution: Trained the model with diverse datasets covering different times of day, weather conditions, and camera angles typical in Saudi Arabia. Implemented confidence thresholds and multiple frame validation to reduce false positives.

Balancing Gamification with Fitness Goals

Ensuring the game elements enhanced rather than distracted from fitness objectives required careful design. Solution: Integrated step count requirements into challenges so users couldn't complete tasks without meaningful physical activity, ensuring health benefits remained central to the experience.

On-Device Performance Optimization

Running ML models on iPhone hardware while maintaining smooth UI performance required optimization. Solution: Implemented efficient model quantization, background processing for non-critical tasks, and lazy loading strategies to minimize memory footprint and battery impact.

Key Learnings
